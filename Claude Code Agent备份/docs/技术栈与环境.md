## 技术栈与环境

- **语言**: Java (>= 17 LTS，最低兼容 1.8，推荐21 LTS以使用Virtual Threads)
- **Web框架**: Spring Boot 3.x
- **数据库/ORM**: MySQL, Spring Data JPA/Hibernate, MyBatis, MyBatis Plus
- **构建/测试/质量**:
  - **构建**: Apache Maven (3.9+) 或 Gradle (8.0+, Kotlin DSL优先)
  - **测试**: JUnit 5 (Jupiter), Mockito, AssertJ, Spring Boot Test
  - **代码规范**: Spotless (Google Java Format或Palantir格式), Checkstyle
  - **静态检查**: SpotBugs, Error Prone (JDK编译期检查), ArchUnit (架构测试)
- **常用工具**:
  - **开发工具**: IntelliJ IDEA, VS Code
  - **数据管理工具**: DataGrip, Redisee
  - **远程工具**: XPipe
  - **终端工具**: iTerm2, zsh
  - **接口调试工具**: Apifox
  - **笔记**: Obsidian, 有道云笔记
  - **通讯工具**: 微信, 企业微信, 飞书, Telegram
